Project: This Portfolio

This portfolio website was built from the ground up to be a dynamic and engaging showcase of my skills. It was designed to be lightweight, fully responsive, and to reflect a modern, tech-focused aesthetic that aligns with my game development work.

Key Skills Demonstrated:

  • HTML5: Used semantic elements like `header`, `footer`, `main`, `section`, and `article` to structure the content logically, improving both accessibility for screen readers and search engine optimization (SEO).
  • CSS3: Implemented a fully responsive design using Flexbox and CSS Grid for layout, ensuring the site is usable on all devices. All animations, including the header flicker, title glitches, and navigation hover effects, were created with pure CSS using `@keyframes` and transitions.
  • JavaScript (ES6): Wrote clean, modular JavaScript to handle all interactive elements. This includes the script that highlights the active navigation link, the controlled flicker effect on the logo, the expandable image lightbox, and the client-side validation for the contact forms.

Project: GameDev Weekly - Newsletter Signup Form

A fully functional, standalone sign-up form for a fictional newsletter. This project demonstrates proficiency in creating structured forms, applying custom styling with pure CSS, and implementing client-side validation with JavaScript to provide immediate user feedback.

Key Skills Demonstrated:

  • Styled Form Controls: Implemented custom styling for all form elements, including text boxes, radio buttons, and checkboxes, to move beyond default browser styles and create a unique, aesthetically pleasing user interface that fits the cyberpunk theme.
  • Client-Side Validation: JavaScript is used to verify that required fields (First Name and Last Name) are filled before allowing the form to be submitted. This prevents incomplete data from being sent to the server and improves the user experience.
  • Custom Alert System: Instead of using the default browser `alert()` for validation messages, a custom-designed modal was created and triggered with JavaScript. This provides a more polished and professional user experience that doesn't break the site's aesthetic.
  • Event Handling: The page demonstrates event handling by using the `onload` event to trigger a welcome message when the page is first loaded, and the `onsubmit` event to trigger the validation logic.
